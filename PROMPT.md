## 2. 技术栈要求
- 框架：React 18 + Vite
- 样式：Tailwind CSS（严格使用响应式：手机端默认样式，电脑端用 `md:` 或 `lg:` 前缀覆写）

## 3. 响应式布局规则（非常重要）
- **手机端（<768px）**：单列布局，底部Tab导航栏，列表单列显示，类似微信小程序风格。
- **电脑端（≥768px）**：左侧侧边栏导航（固定在左侧），右侧主内容区，列表采用2-3列网格展示，类似闲鱼网页版风格。
- 采用 Mobile First 原则写样式（先写手机版，再写 `md:` 电脑版）。

## 4. 核心页面规划
1. **登录页**：模拟统一身份认证，居中卡片。
2. **首页/广场**：包含“技能广场”和“需求广场”两个Tab。卡片必须展示：头像、姓名、技能/需求标签、价格/预算、信用星级。
3. **发布页**：包含“发布技能”和“发布需求”两个表单。字段必须严格对应 CONTEXT.md 中的字段（如 skill_tag, service_price, budget_amount 等）。
4. **订单详情**：展示状态流转（待确认->进行中->待验收->已完成），包含操作按钮。
5. **个人中心**：展示信息、信用分(score+level)、钱包余额。

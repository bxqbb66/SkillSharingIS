# 第八步开发任务：视觉与体验全面打磨（UI/UX 升级）

核心业务逻辑已全部完成，现在需要将项目从“功能可用”升级为“视觉专业”，达到可展示的毕业设计/作品集标准。请进行以下全局优化：

1. **吉大品牌配色与全局主题**：
   - 设定全局主色调。请使用吉林大学的标准色系（建议主色：吉大蓝 #003366 或 校徽红 #A4212E），辅以中性灰和背景色 #F5F5F5。
   - 确保所有按钮、Tab高亮、链接等交互元素使用统一的主色调。
   - 全局字体规范：标题、正文使用合理的字重和大小对比，增加层次感。

2. **骨架屏加载状态**：
   - 目前页面切换和数据渲染是瞬间出现的，显得生硬。
   - 在首页卡片加载、详情页内容加载、个人中心加载时，加入 1-2 秒的骨架屏动画，然后再渲染真实数据，模拟真实网络请求的流畅感。

3. **优美的空状态**：
   - 当搜索无结果、工单列表为空、没有评价时，不要只显示干巴巴的文字。
   - 请设计居中的空状态提示：一个简约的 SVG 插画（或大图标）+ 引导文字（如“暂无工单，去广场看看吧”），让空页面也不显得简陋。

4. **图片资源升级**：
   - 淘汰目前所有灰色的占位图框！
   - 用户头像：使用类似 `https://api.dicebear.com/7.x/avataaars/svg?seed=xxx` 或 `https://i.pravatar.cc/150?img=xxx` 的免费随机头像 API，根据用户 ID 生成不同的生动头像。
   - 技能/需求封面：使用 `https://picsum.photos/seed/xxx/400/300` 这种免费随机图片 API 作为卡片的封面图，让广场和详情页看起来图文并茂。

5. **微交互与动效**：
   - 卡片 Hover 效果：电脑端鼠标悬停在卡片上时，加上轻微的上移和阴影加深效果（transition: all 0.3s）。
   - 按钮点击反馈：点击主按钮时，加上轻微的缩放或变色反馈。
   - 页面切换：如果容易实现，加入简单的淡入动效。

请全局排查并应用以上优化，让整个应用焕然一新。
